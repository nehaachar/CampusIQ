# CampusIQ
An AI-powered chatbot for college websites that intelligently answers student queries using documents (PDFs, websites) with RAG + Gemini LLM.

Problem Statement:

College websites are often hard to navigate, and students struggle to find accurate information about:
Admissions
Exams
Scholarships
Events

This leads to time wastage and poor user experience.

Solution:

CampusIQ uses a GenAI-based retrieval pipeline to:

Convert documents into embeddings for semantic search
Retrieve relevant content using RAG (Retrieval-Augmented Generation)
Perform context injection into the Gemini LLM
Generate accurate, human-like responses using LLM reasoning



CampusIQ uses a GenAI-based agentic pipeline to:

Process PDFs and websites using Gemini file handling
Route queries using a tool-calling agent
Retrieve relevant information from documents
Generate context-aware responses using the Gemini LLM

Goal: Improve Information access and enhance student experience on the college website.
